# 办公软件安装与授权

## 标准软件清单

公司标准办公软件通常包括：

- Microsoft 365 Apps 或 WPS Office。
- Outlook、Teams、OneDrive 或企业 IM 工具。
- Chrome 或 Edge 浏览器。
- PDF 阅读器。
- VPN 客户端。
- 终端安全、设备管理和补丁客户端。
- Git、VS Code、JetBrains Toolbox 等开发工具，视岗位需要安装。

非标准软件需要业务说明、直属经理审批，必要时还需要安全或合规审批。

## 软件中心安装流程

1. 连接公司办公网络或 VPN。
2. 打开公司软件中心或设备管理门户。
3. 搜索需要的软件名称。
4. 点击安装并等待下载、安装和配置完成。
5. 安装完成后重启软件；必要时重启电脑。
6. 使用公司账号登录并完成授权激活。

如果软件中心未显示目标软件，请确认设备是否已纳入公司管理。

## 安装失败处理

常见原因：

- 磁盘空间不足。
- 网络中断或 VPN 断开。
- 旧版本未卸载干净。
- 安装包被安全策略阻止。
- 当前用户缺少管理员权限。

建议步骤：

1. 确认系统盘至少有 10 GB 可用空间。
2. 重启电脑后再次从软件中心安装。
3. 卸载旧版本并清理残留插件。
4. 确认终端安全客户端运行正常。
5. 记录错误代码和失败截图。

## 授权与许可证问题

常见现象：

- Office 提示未激活或只读模式。
- Teams 或 OneDrive 无法使用公司账号登录。
- 专业软件提示许可证已用尽。
- 离线时间过长导致授权状态失效。

处理建议：

- 确认使用公司账号登录，而不是个人账号。
- 连接公司网络或 VPN 后重新打开软件。
- 对共享许可证软件，确认是否仍有可用席位。
- 对岗位授权软件，确认员工是否在授权名单中。

## 何时创建工单

请创建 IT 工单并提供：

- 软件名称、版本和安装来源。
- 电脑主机名、系统版本和当前网络环境。
- 错误截图、错误代码和失败时间。
- 是否需要紧急安装或涉及业务阻塞。

以下情况应直接创建工单：

- 软件中心安装连续失败。
- 需要管理员权限或白名单放行。
- 标准授权无法激活。
- 需要申请非标准软件或额外许可证。
