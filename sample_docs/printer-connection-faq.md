# 打印机连接 FAQ

## Windows 添加网络打印机

1. 连接公司办公网络或 VPN。
2. 打开“设置”>“蓝牙和设备”>“打印机和扫描仪”。
3. 选择“添加设备”。
4. 如果未自动发现，选择“手动添加”。
5. 输入打印服务器或打印机共享路径，例如 `\\print-server\office-3f-printer`。
6. 按提示安装驱动并打印测试页。

如果提示需要管理员权限，请创建 IT 工单安装标准驱动。

## macOS 添加网络打印机

1. 连接公司办公网络或 VPN。
2. 打开“系统设置”>“打印机与扫描仪”。
3. 点击添加打印机。
4. 在 IP 或 Windows 标签中输入打印机地址或共享路径。
5. 协议优先选择 AirPrint、IPP 或公司文档指定的协议。
6. 选择对应驱动后添加并打印测试页。

如果无法找到驱动，优先使用公司软件中心提供的打印驱动包。

## 常见问题

### 打印机显示离线

- 确认电脑已连接公司网络。
- 检查打印机电源、网线和面板状态。
- 取消队列中的失败任务后重新打印。
- 重启打印机和电脑后再次测试。

### 打印任务卡住

Windows 可尝试清空打印队列并重启 Print Spooler 服务：

```powershell
Restart-Service Spooler
```

macOS 可在打印队列中删除失败任务，再重新提交打印。

### 打印乱码或格式错乱

- 确认选择了正确的打印机型号驱动。
- 对 PDF 文件尝试“作为图像打印”。
- 检查纸张尺寸、双面打印和黑白彩色设置。

### 无法使用扫描功能

扫描功能可能需要单独安装扫描驱动，或需要将电脑加入允许扫描的网络范围。

## 何时创建工单

请创建 IT 工单并提供：

- 打印机名称、位置和楼层。
- 电脑系统版本。
- 错误截图或打印机面板提示。
- 是否只有本人受影响，还是同区域多人受影响。

以下情况应直接创建工单：

- 打印机硬件报错、缺纸缺粉以外的故障码。
- 需要管理员权限安装驱动。
- 多人无法连接同一台打印机。
- 打印服务器共享路径不可访问。
