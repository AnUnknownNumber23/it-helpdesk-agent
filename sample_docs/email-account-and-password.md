# 邮箱账号与密码支持

## 新员工邮箱开通

新员工邮箱通常由 HR 入职流程触发创建。邮箱创建完成后，IT 会同步分配初始登录方式和 MFA 注册说明。

开通前需要确认：

- 员工姓名、工号、部门和直属经理。
- 入职日期和办公地点。
- 是否需要加入部门邮件组或共享邮箱。
- 是否需要访问历史项目邮箱或特殊系统通知组。

如果入职当天仍未收到邮箱信息，请由直属经理或 HR 创建 IT 工单。

## 密码重置

员工忘记密码或密码过期时，应优先使用公司自助密码平台处理。

建议流程：

1. 打开自助密码平台。
2. 使用员工号或公司邮箱账号验证身份。
3. 通过 MFA、短信或预留备用邮箱完成校验。
4. 设置符合公司复杂度要求的新密码。
5. 使用新密码重新登录邮箱、VPN、办公电脑和相关系统。

密码修改后，请及时更新手机邮箱、Outlook、VPN、Git 客户端等保存过旧密码的应用。

## MFA 设备丢失或更换手机

如果员工更换手机、手机丢失、Authenticator 应用被卸载，可能无法完成 MFA 验证。

处理建议：

- 如果仍有备用 MFA 方式，先登录安全信息页面重新绑定新设备。
- 如果没有任何可用 MFA 方式，需要创建 IT 工单重置 MFA。
- 涉及手机丢失时，应同步通知安全团队评估账号风险。

工单中应包含员工账号、联系电话、直属经理审批信息，以及是否需要紧急恢复访问。

## 账号锁定

账号锁定通常由多次错误密码、旧设备持续重试或异常登录触发。

常见触发源：

- 手机邮箱仍保存旧密码。
- VPN 客户端自动重连使用旧密码。
- 浏览器保存的旧密码自动填充。
- 离职或转岗流程导致账号状态变更。

处理步骤：

1. 停止继续尝试登录。
2. 检查所有设备并清理旧密码。
3. 等待自动解锁或在自助平台申请解锁。
4. 如果仍被锁定，创建 IT 工单。

## 何时创建工单

请在以下场景创建工单：

- 新员工邮箱未按入职流程创建。
- 自助密码平台无法完成身份验证。
- MFA 丢失且没有备用验证方式。
- 账号反复锁定，无法定位触发设备。
- 邮箱登录成功但无法收发邮件或缺少授权邮箱。
